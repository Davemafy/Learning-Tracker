# Brillo

Brillo is a simple learning tracker that helps users log daily learning activity and stay consistent.

## Features
- Add daily learning entries
- View learning history
- Responsive, mobile-first design

## Tech Stack
- React
- JavaScript
- Tailwind CSS
- (Optional) API / local storage

## Why I built this
I wanted to practice building a real, usable web app instead of isolated components.
